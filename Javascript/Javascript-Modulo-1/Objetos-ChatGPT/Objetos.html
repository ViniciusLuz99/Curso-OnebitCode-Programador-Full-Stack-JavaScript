Objetos

No JavaScript, um objeto é como um recipiente que contém diferentes informações relacionadas a uma coisa específica. Imagine que você tenha uma caixa de brinquedos. Cada brinquedo na caixa possui características diferentes, como cor, tamanho e forma. No JavaScript, você pode pensar em cada brinquedo como um objeto e as características do brinquedo como propriedades desse objeto.

Vamos pegar o exemplo de uma pessoa. Uma pessoa pode ter propriedades como nome, idade e gênero. Para representar essa pessoa como um objeto em JavaScript, você pode criar uma variável e atribuir a ela um valor de objeto. Aqui está um exemplo:

var pessoa = { nome: "João", idade: 25, genero: "masculino" };

Nesse exemplo, criamos um objeto chamado pessoa com três propriedades: nome, idade e genero. Os nomes das propriedades estão à esquerda dos dois pontos (:), e seus valores estão à direita.

Você pode acessar as propriedades de um objeto usando a notação de ponto ou colchetes. Veja como você pode acessar as propriedades do objeto pessoa:

console.log(pessoa.nome); // Saída: "João" console.log(pessoa.idade); // Saída: 25 console.log(pessoa.genero); // Saída: "masculino"

No JavaScript, os objetos também podem ter funções como propriedades. Essas funções são chamadas de métodos. Elas permitem que objetos executem ações. Veja um exemplo:

var pessoa = { nome: "João", idade: 25, genero: "masculino", dizerOla: function() { console.log("Olá, meu nome é " + this.nome); } };

pessoa.dizerOla(); // Saída: "Olá, meu nome é João" Nesse exemplo, o objeto pessoa possui um método chamado dizerOla que registra uma mensagem no console usando o nome da pessoa.

Agora, para o desafio! Vamos criar um objeto representando um carro. O carro deve ter propriedades como marca, modelo e ano. Ele também deve ter um método chamado ligarMotor que registra uma mensagem dizendo "O carro está ligando". Tente criar o objeto e chamar o método ligarMotor.
